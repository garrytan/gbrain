# Shared context — PASA daily organization protocol

The daily schedule every PASA role operates within. Prime any care agent with
this so its recommendations align with the actual day structure (workshop,
hydration, meal, toileting, and handover windows).

| Time | Activities |
|---|---|
| 10:00–10:30 | Welcome residents to the PASA; coffee + music; present the day's programme. |
| 10:30–11:15 | Body-awareness / gentle-movement workshop; **hydration** mid-session (some residents have individual drink preferences recorded in their care plan). |
| 11:20–12:00 | Press-review workshop (newspaper, discussion, participation). |
| 12:00–12:30 | Toileting + handwashing; read the menu; residents set the table; collect meal trolley. |
| 12:30–13:30 | Therapeutic meal. |
| 13:30–14:30 | Clear/clean; quiet time in the lounge (music, magazines, board games). |
| 14:30–16:00 | Workshop; **hydration** checks; snack trolley collected ~15:30. |
| 16:30–17:00 | Snack; review the day with residents. |
| 17:00–17:30 | Toileting; accompany residents back to their units. |
| 17:30–18:00 | Complete handover documentation; team handover. |

Key recurring touchpoints an agent should respect: **hydration** (mid-morning +
afternoon), **toileting windows** (12:00–12:30, 17:00–17:30), **therapeutic
meal** (12:30–13:30), and the **end-of-day handover** (17:30–18:00).
