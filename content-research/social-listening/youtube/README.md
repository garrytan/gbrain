# YouTube Social Listening

## Purpose

Raw data from YouTube crawls. YouTube is the second-largest search engine and a key source for understanding what content audiences consume and what questions they still have.

## What Data Lives Here

- Video metadata (title, description, tags, view count, like count, comment count)
- Top-level comments and reply threads
- Channel metadata (subscriber count, upload frequency)
- Search result data for target queries

## File Naming Convention

```
YYYY-MM-DD-<query-or-channel>.json
```

Examples:
- `2026-05-30-ai-tools-review.json`
- `2026-05-29-seo-tips-2026.json`
- `2026-05-28-channel-hubspot.json`

## Data to Capture

- Top 20-50 search results per target query
- Top 100 comments per video (sorted by relevance)
- Video duration, publish date, and engagement rate

## Notes

- YouTube comment sections often contain detailed pain points and questions
- Video titles and descriptions are goldmines for keyword research
- Track competitor channels for content gap analysis
