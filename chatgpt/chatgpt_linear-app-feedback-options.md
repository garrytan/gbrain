## User

can i use linear.app to get feedback from users diretly?

---

## Assistant

Yes — but **not directly from end users out of the box** in the way a dedicated feedback widget (like Canny or Userback) works.

**What Linear *does* offer:**

- **Customer Requests:** Linear has a built-in *Customer Requests* feature that lets you attach real user feedback to issues and projects so your team can see exactly what customers said and prioritize accordingly. You can link feedback to issues, filter by customer attributes, and track demand. citeturn0search8turn0search11

- **Integrations for direct user input:** Linear doesn’t provide a public feedback widget itself, but you can use integrations with tools like:
  * **Featurebase** — public feedback boards where users can vote and comment, synced into Linear issues. citeturn0search1  
  * **Canny**, **Userback**, or similar feedback collection services — embed a feedback widget on your site/app and automatically push submissions to Linear. citeturn0search5turn0search18  
  * **Intercom**, **Zendesk**, **Front**, etc. — capture user comments/bug reports from support chats and turn them into Linear issues with a linked request. citeturn0search8turn0search9

**Limitations:**
- Linear alone isn’t a self-serve user feedback portal; it’s primarily an internal issue/feature tracking system. Teams generally combine it with external feedback tools. citeturn0search0

**Bottom line:** yes, you can **surface and manage real user feedback in Linear**, but you’ll typically need to set up an **integration or external feedback board** if you want users to submit feedback directly.
