## The logo should look like this conceptually

**Primary lockup:**  
`[stroke-circle icon]  Jovie`

Use **Jovie**, with an uppercase **J**. Avoid `jovie`. Avoid `JOVIE`.

The uppercase **J** gives you immediate separation from the lowercase childcare brand, while keeping the name human and premium. All caps would feel colder and more enterprise; lowercase would drift too close to their territory.

## Icon direction

Use the existing icon idea, but polish it into a system:

**A perfect stroked circle, no fill, rounded caps, one inner orbital stroke.**

The mark should feel like:

- momentum
- an intelligent loop
- an agent/flywheel
- software precision
- calm motion

It should avoid:

- a literal **J**
- a smiley/face
- a kid/family cue
- a heart
- a sun/horizon shape
- a filled donut
- bubbly daycare softness

Airbnb’s Bélo is the correct standard here: a simple symbol at the center of the identity, understandable across language, and simple enough to draw from memory. That is the test for Jovie too. citeturn358888view3

## Wordmark direction

The wordmark should be **custom-drawn from a modern grotesk**, then edited to match the icon geometry. Do not ship an off-the-shelf font as the final logo.

Best starting references:

- **Söhne**
- **ABC Diatype**
- **Geist**
- **Inter Display**
- **Avenir Next**
- **Circular**
- **Matter**
- **Neue Haas Grotesk**

The final wordmark should sit around **SemiBold / 600**, with the same optical thickness as the icon stroke.

### Letterform rules

**J**  
Make this the hero. It should be a clean uppercase J with a smooth hook and subtle rounded terminal. The hook radius should echo the icon’s curve radius.

**o**  
Nearly circular, with an open, calm counter. It should visually rhyme with the icon’s outer ring.

**v**  
Sharp enough to feel like software, softened enough to match the icon. Avoid a bubbly rounded-bottom v.

**i**  
Use a small, precise dot. The dot should feel engineered, not playful. A perfectly circular dot is fine if it matches the icon stroke; a slightly squared optical dot may feel more SaaS.

**e**  
Open aperture, clean horizontal bar, no quirky angled cuts. The counter should relate to the **o** and the icon’s inner negative space.

## Lockup proportions

Use these as the first design spec:

- Icon height: **1.00**
- Wordmark cap/x-height: **0.62–0.68** of icon height
- Gap between icon and wordmark: **0.30–0.36** of icon width
- Icon stroke: **10–12%** of icon diameter
- Wordmark stem thickness: visually equal to icon stroke
- Kerning: tight but not compressed; manually tune **Jo**, **ov**, **vi**, **ie**
- Primary logo: black on white
- Reverse logo: white on black
- Accent: `#635AFF`, mainly for icon/app contexts

Linear’s own brand guidance is a good SaaS model: use the wordmark when space allows, use the logomark in tight layouts, and prefer monochrome usage. citeturn358888view2

## Color system

Default identity should be:

- **Jovie Black**: near-black, not pure harsh black; around `#09090B`
- **Jovie White**: `#FFFFFF`
- **Jovie Violet**: existing `#635AFF`
- **Soft Surface**: `#F7F7FA`
- **Dark Surface**: `#111114`

Do **not** make the full wordmark purple by default. That will feel less premium. Use purple as an active product signal, app icon color, hover state, or hero accent.

## What makes it world-class

The winning version will feel like it was drawn from one geometric system:

The icon circle, the **o**, the **e** counter, the **J** hook, the **i** dot, and the stroke weight should all look mathematically related.

That is the difference between “logo with a font next to it” and a real brand mark.
