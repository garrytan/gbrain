## User

Hey fam, I'm trying to send a series of photos and videos to Geico via email. If I send these as like a shareable link or something, are they gonna be able to download it, or do I have to like manually send these all in like 25 different emails?

---

## Assistant

Mr. White, **send a shareable link**, ideally from **Google Drive, Dropbox, iCloud, or OneDrive**. Do **not** split it into 25 emails unless GEICO specifically asks.

### Best setup

Create one folder:

**GEICO Claim Photos + Videos – [Your Claim Number]**

Inside:

- Photos
- Videos
- Police report photo/PDF
- Insurance info
- Driver info
- Any repair/tow documents

Then set permissions to:

> **Anyone with the link can view/download**

Before sending, open the link in a private/incognito browser to confirm it works without logging in.

### Use Google Drive if possible

Google Drive is usually easiest because claim adjusters can open it in a browser and download files individually or as a ZIP.

Avoid links that require:
- Apple ID login
- Requesting access
- Expiring download permissions
- “Only people invited can view”

### Email text to GEICO

:::writing{variant="email" id="42861" subject="Claim photos and videos – Claim #[CLAIM NUMBER]"}
Hi,

I’m sending photos and videos related to claim #[CLAIM NUMBER].

The files are available here:

[PASTE SHAREABLE LINK]

The folder includes photos, videos, and supporting documentation from the incident. The link is set so anyone with the link can view and download the files.

Please let me know if you have any trouble accessing it.

Best,  
Tim White
:::

### One extra move

Attach **3–5 key photos directly** to the email too: wide shot, damage closeups, other car, license plate, police report/card if you have it. Then the link has everything else.

## Assistant

Mr. White, that **iCloud Photos share link should work**, but it is a slightly riskier choice than Google Drive because some corporate claim systems block iCloud links or make downloads annoying.

Use it only after testing this:

1. Open the link in **Safari/Chrome private browsing**
2. Make sure it opens **without your Apple ID**
3. Confirm there is a visible **download option**
4. Make sure the videos play/download
5. Confirm the album includes everything you want GEICO to see

If it passes that test, send it.

Email line:

> Photos and videos are available here: https://share.icloud.com/photos/0eayDLDKkS4_fQkfqgb2pfbQw  
> The link should allow viewing and downloading without requiring an Apple ID. Please let me know if you have any trouble accessing it.

Safer version: keep the iCloud link, but also upload the same files to **Google Drive** and send both links. GEICO is more likely to handle Drive cleanly.
