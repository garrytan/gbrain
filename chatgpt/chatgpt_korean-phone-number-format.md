## Assistant

A Korean phone number format typically follows this pattern:

1. **Mobile Numbers:** 
   - Format: 010-XXXX-XXXX 
   - Example: 010-1234-5678

2. **Landline Numbers:**
   - Format: 02-XXXX-XXXX for Seoul 
   - Example: 02-1234-5678
   - For other regions: 0XX-XXXX-XXXX 
   - Example: 031-1234-5678 for Gyeonggi Province

For international dialing:
- Format: +82-10-XXXX-XXXX for mobile 
- Example: +82-10-1234-5678
- Drop the leading zero when dialing internationally.
