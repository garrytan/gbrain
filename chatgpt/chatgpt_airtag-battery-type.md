## User

What kind of battery does an AirTag take?

---

## Assistant

An AirTag uses a **CR2032** coin cell battery (3V lithium).  

- It’s a **standard, user-replaceable** coin battery.  
- Make sure to install it **positive (+) side up**.  
- Some AirTags can be finicky with **CR2032 cells that have a bitterant coating** (often used in child-safe versions), so a plain CR2032 usually works best.

---

## User

So 2450 is no good?

---

## Assistant

Correct, Mr. White — **CR2450 won’t work for an AirTag**.

- AirTag is specifically designed for a **CR2032**.
- A **CR2450 is larger and thicker**, so it **doesn’t fit properly** and isn’t recommended even though it’s also 3V.
- Stick with a **plain (non-bitterant-coated) CR2032** for best reliability.
