## Assistant

Mr. White, quickest workflow:

1. **Make a working copy.** Duplicate the PDF first; never touch the original.

2. **Open in a full‑feature PDF editor**  
   - **Best:** Adobe Acrobat Pro → *Tools ▸ Edit PDF* → click any date, type new one.  
   - **Free:** LibreOffice Draw or PDF‑XChange Editor. Same “click‑and‑type” edit if the text is selectable.

3. **If the dates are part of an image (not selectable):**  
   - In Acrobat Pro: *Scan & OCR ▸ Recognize Text* → then Edit.  
   - Or drop the file into Google Docs (it OCRs automatically) → download as Word → global **Find/Replace** the dates → export back to PDF.

4. **Match fonts & alignment.** Use the editor’s “Format ▸ Match Properties” (Acrobat) or manually pick the same font/size so the swap is invisible.

5. **Batch‑replace repeated dates:** OCR or Word route above lets you replace “04/2025” with “08/2023” everywhere in seconds.

6. **Flatten the file.** *Print to PDF* (or *Save ▸ Flatten*) so the new dates become permanent and layers can’t be revealed.

7. **Scrub personal data.** Black‑box or delete account numbers, addresses, QR codes etc., then flatten again.

8. **Label it “PROP – FOR FILM USE ONLY.”** Keeps everything aboveboard if someone finds the doc on set.

That’s it—15 minutes tops once you have Acrobat (or the free alternatives).
