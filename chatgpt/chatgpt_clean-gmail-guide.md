## Assistant

Mr. White, here’s a **fast, high-leverage way to clean up Gmail** without wasting time.

---

## 1. Delete in bulk (5 minutes)
Use search operators, then **Select all → Select all conversations → Delete**.

- Promotions:  
  `category:promotions`
- Social:  
  `category:social`
- Old mail (example):  
  `before:2023/01/01`
- Large emails:  
  `larger:5M`
- Unread junk:  
  `is:unread`

This alone usually removes **60–80%** of clutter.

---

## 2. Kill repeat senders (unsubscribe once, forever)
Search by sender, then unsubscribe and delete all.

- Example:  
  `from:substack.com`  
  `from:medium.com`  
  `from:linkedin.com`

Tip: If Gmail shows **“Unsubscribe”** at the top, use it—faster and cleaner.

---

## 3. Use filters to prevent future mess (10 minutes)
Create filters that auto-label, archive, or delete.

Example filters:
- Newsletters → **Skip Inbox + Label**
- Receipts → **Label “Receipts”**
- Notifications → **Archive**

How:
1. Open an email → 3 dots → *Filter messages like this*
2. Choose action → Apply to matching conversations

---

## 4. Turn off Inbox noise
Settings → Inbox:
- Inbox type: **Default**
- Categories: **Only Primary** (optional Promotions if needed)
- Importance markers: **Off**
- Nudges: **Off**

This reduces visual clutter and cognitive load.

---

## 5. One-touch maintenance habit
Once a week (2 minutes):
- Search `is:unread`
- If it’s not actionable → delete or archive immediately

Inbox zero is not the goal. **Low friction is.**

- **Unroll.Me** – newsletter cleanup (use cautiously with privacy)
