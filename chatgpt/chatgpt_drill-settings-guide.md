## Assistant

Here’s how to use the settings on that DeWalt, Mr. White:

### 1. Torque ring (the numbered collar behind the chuck)
- The numbers (1,2,3… up to ~15) control **how hard it can twist before slipping**.
- **Low number = weak, slips early** → for small screws / soft materials.
- **High number = strong, slips late** → for bigger screws / harder materials.
- For driving screws:  
  1. Start around **5–7**.  
  2. If the clutch slips before the screw is seated, go up a couple numbers.  
  3. If it buries the screw too deep, go down.

At the end of the numbers you’ll have a **drill bit icon**:
- That mode **disables the clutch** → full power, no slip.  
- Use this only for **drilling holes**, not for driving screws.

### 2. Top speed switch (usually “1 / 2” on top)
- **1 = low speed, high torque**  
  - Big screws, large drill bits, metal, or when the drill is struggling.
- **2 = high speed, lower torque**  
  - Small screws, small drill bits, light work in wood/drywall.

### 3. Direction switch (above the trigger)
- Push to the **left** (looking from behind) → **forward** (drives screw in).  
- Push to the **right** → **reverse** (backs screw out).  
- Centered → **locked** (safety).

### 4. Trigger
- Variable speed: the more you squeeze, the faster it spins.  
- Feather it to start slow so bits don’t walk and screws don’t strip.

#### Quick recipes
- **Driving wood screws into studs:**  
  - Speed **1**, torque **8–12**, screw icon mode (not drill icon), start lower and adjust.
- **Drilling wood/drywall:**  
  - Speed **2**, **drill icon** on the ring.
- **Drilling metal:**  
  - Speed **1**, **drill icon**, light pressure, use cutting oil if you have it.

If you tell me what you’re about to drill into (stud, concrete, furniture panels, etc.), I can give you exact numbers to start with.
