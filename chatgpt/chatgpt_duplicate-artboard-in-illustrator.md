## Assistant

To duplicate an artboard in Adobe Illustrator, follow these steps:

1. Open your Illustrator document.
2. Select the Artboard Tool (Shift + O).
3. Click on the artboard you wish to duplicate.
4. In the Properties panel (or Control panel at the top), you'll see an option to "Duplicate Artboards." Click it.
5. The duplicated artboard will appear next to the original. You can click and drag to reposition it as needed.

Alternatively, you can also drag the artboard with the Artboard Tool while holding down the Alt (Windows) or Option (Mac) key to create a duplicate.

## Assistant

To add a drop shadow in Adobe Illustrator, follow these steps:

1. Select the object to which you want to add a drop shadow.
2. Go to the "Effect" menu in the top menu bar.
3. Choose "Stylize" > "Drop Shadow..."
4. In the Drop Shadow dialog box, you can adjust settings like:
   - **Mode:** Blend mode for the shadow.
   - **Opacity:** Transparency level of the shadow.
   - **X Offset:** Horizontal distance from the object.
   - **Y Offset:** Vertical distance from the object.
   - **Blur:** Softness of the shadow edges.
   - **Color:** Shadow color.

5. Click "OK" to apply the drop shadow.

You can preview the changes as you adjust the settings if you have the "Preview" box checked in the dialog box.

## Assistant

To adjust opacity in Adobe Illustrator:

1. **For Individual Objects:**
   - Select the object whose opacity you want to adjust.
   - In the Properties panel (or the Control panel at the top of the workspace), you'll find an opacity field.
   - Enter a percentage or use the slider to set the desired opacity level.

2. **For Groups or Layers:**
   - If you want to adjust the opacity of a group of objects or an entire layer, first select all objects in the group or the layer you want to adjust.
   - Use the same opacity field in the Properties or Control panel to adjust the opacity for the selection.

Opacity ranges from 0% (completely transparent) to 100% (completely opaque).
