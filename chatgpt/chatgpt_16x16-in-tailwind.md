## Assistant

To make something **16×16** in Tailwind CSS, use the built-in spacing scale for width and height:

```html
<div class="w-16 h-16 bg-gray-200"></div>
```

- `w-16` → width: 4rem (64px)
- `h-16` → height: 4rem (64px)

If you need **16px × 16px**, use:

```html
<div class="w-4 h-4 bg-gray-200"></div>
```

Tailwind’s spacing scale:  
• `4` = 1rem (16px)  
• `16` = 4rem (64px)
