Refactor some large sections of our codebase to make it easier for us to move quickly & add new features.

Faster loading times, Dark Mode, and more.

Improved load times.

General code refactors

Fixes an issue with images from Twitter not displaying correctly in some cases.
Fixes an issue with images not displaying in the CRM in some cases.
Fixes an issue with pagination not working properly on the CRM in some cases.
Improved loading screens on first visit
Fixed a bug in a new unreleased API endpoint
Add instructions for contacts not saved already
Fixed various bugs & improved performance

Chrome Extension, List Managment, & Bug Fixes
This release adds search to dropdown menus, making it much easier to add a contact to a list when you have a large number of lists. We've alo brough all the functionality of the contact sidebar to the chrome extension so you can add social media profiles to lists, modify their contact information, or leave a note, without leaving their social profile.

New Features
Added the ability to add contacts to lists from a social media page.
Added the ability to search for lists in the dropdown menu.
Added the ability to add a note to a contact from a social media page.
Added the ability to modify a contact's information from a social media page.
Fixes & Improvements
Removed redudant navigation elements
Conformed settings page to new design
Added search to dropdown menus
Added breadcrumbs to sidebar
Chrome extension now has all the functionality of the CRM sidebar
Fixed various bugs & improved performance

- Fixed an issue where importing a contact from the dashboard & again from the chrome extensino created a duplicate contact
- Fixed an issue that caused the Chrome Extension to crash when navigating between profiles
- Fixed an issue that caused Instagram imports to fail on some new accounts
