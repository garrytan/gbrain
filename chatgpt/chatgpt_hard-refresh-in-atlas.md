## Assistant

Mr. White, a **hard refresh in the Atlas browser** forces the page to reload from the server and clears cached assets for that page.

### **How to hard-refresh**
Try any of these:

- **Windows/Linux:**  
  **Ctrl + Shift + R** or **Ctrl + F5**

- **Mac:**  
  **⌘ + Shift + R**

If the site still serves cached data, manually clear cache:

1. Open Atlas settings
2. Clear site cache or “Clear browsing data”
3. Reload the page

If you want, I can guide you step-by-step for your OS.
