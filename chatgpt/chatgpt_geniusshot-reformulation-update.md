  FDA rounding rules let companies round **>50 kcal to the nearest 10** (e.g., **96 → 100**). They can also show **0 g** for carbs/fat if each is **<0.5 g**. So tiny amounts + protein rounding can push a 92–98 kcal true value to a **100-kcal label** with **0 g** fat and carbs still showing. citeturn5search0turn5search12
