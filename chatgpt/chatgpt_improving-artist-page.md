## Assistant

Mr. White — this page is super barebones right now (name + “Artist” + notification opt-in + socials). You’ll get a lot more follows/streams if you turn it into a **1-screen “what to do next” hub**.

## High-impact upgrades (in order)

### 1) Add a clear 1-line positioning
Right now people don’t know what you sound like or why to care.
- Example: **“Alt-pop / dance — LA. For fans of ___ / ___.”**
- Or: **“Singer-songwriter making ___.”**

### 2) Put your #1 goal as the first button
Pick one primary outcome:
- **Listen to the new single** (best default)
- **Pre-save / Save on Spotify**
- **Watch the newest video**
- **Text list / email list** (if you’re doing frequent drops)

### 3) Make the notification capture worth it
“Never miss a release” is fine, but add a reason + expectation:
- “**Text me when I drop music + shows. 1–2/month.**”

### 4) Add 5–7 key links max (with specific labels)
People won’t click vague links. Use action labels:
- **Listen: Latest single**
- **Watch: New video**
- **Spotify / Apple Music**
- **Tour / Shows**
- **Merch**
- **Press kit / EPK**
- **Contact / Booking**

### 5) Add social proof (even tiny)
A single line helps conversions:
- “**Played ___ / Featured on ___ / ___ streams**”
- Or: “**New EP out now**” / “**On tour**”

### 6) Make “Tip” optional, not prominent
Tips work best after value. Put it lower, or label it warmly:
- “**Support (tip jar)**”

### 7) Track clicks
If Jovie supports it, add UTM links to Spotify/YouTube so you know what’s working.
