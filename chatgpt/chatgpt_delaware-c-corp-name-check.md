Delaware’s **Division of Corporations** provides a *Name Availability Search* tool you should use to check if your desired C-Corp name is available before you file:

🔹 The system will tell you *available* or *not available*.

### 🔎 2) Alternative: General Entity Search
You can also use the **Business Entity Search** to look up existing Delaware entities (whether active or inactive). This helps you see *all similar or identical names* already registered. citeturn0search2

### 📝 Naming Rules to Keep in Mind
A name must:
- Be **distinguishable** from other Delaware entity names.  
- Include a proper **corporate suffix** (like “Inc.” or “Corp.”).  
- Avoid prohibited or restricted words under Delaware naming rules. citeturn0search3

### 📌 Optional: Reserve the Name
If the name *is* available and you’re not ready to file immediately, you can reserve it with the state for up to **120 days** for a fee. citeturn0search3

### ⚠️ Trademark & Other Checks
Name availability at the state level doesn’t guarantee you can use it commercially. You should also:
- Do a **USPTO trademark search** to avoid federal trademark conflicts.
- Check domain name availability if you need a web presence.
