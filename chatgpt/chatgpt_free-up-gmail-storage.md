## 1. Delete large emails first (highest ROI)
- In Gmail search bar, run:
  - `larger:10M`
  - `larger:5M`
- Sort by size → bulk delete emails with attachments (PDFs, videos).
- Empty **Trash** immediately (deleted emails still count until emptied).

## 2. Clean Spam & Trash (quick win)
- Go to **Spam** → *Delete all spam messages now*
- Go to **Trash** → *Empty Trash now*
- This alone often frees **hundreds of MBs to multiple GBs**.

## 3. Remove old bulk senders
Search and mass delete:
- `from:noreply@`
- `from:no-reply@`
- `unsubscribe`
- `category:promotions`
Then empty Trash.

## 4. Check total storage usage (don’t guess)
- Visit **one.google.com/storage**
- See exact split between **Gmail / Drive / Photos**
- If Gmail isn’t the main offender, don’t waste time there.

## 5. Drive & Photos spillover (often the real issue)
- **Google Drive**: sort by **Storage used** → delete large files
- **Google Photos**:
  - Delete videos
  - Empty Trash
  - If still needed, switch to **Storage saver** (compresses old photos/videos)

## 6. Detach but keep files (advanced)
- Download important attachments
- Delete the email
- Re-upload to Drive if needed (net storage may still drop if Photos was the sink)

## What *doesn’t* help
- Archiving emails ❌ (still counts)
- Labels ❌
- Read/unread status ❌
