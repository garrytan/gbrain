## User

My ac is not working. It’s on the window

---

## Assistant

Mr. White, first check the basics in this order:

## 1. Confirm the unit is actually set to cool
- Mode: **Cool**, not Fan/Eco/Dry.
- Temp: set it to **60–65°F** temporarily.
- Fan: set to **High**.
- Wait **3–5 minutes**. Some units delay compressor start.

## 2. Check power reset
- Unplug it for **60 seconds**.
- Plug it back in directly to the wall.
- Press the **reset/test button** on the plug if it has one.
- Avoid extension cords or power strips.

## 3. Check the filter
- Pull the front filter out.
- If it is dusty, rinse it, dry it, and reinstall.
- A clogged filter can make it blow weak or warm.

## 4. Check whether the compressor kicks on
Listen for two sounds:
- **Fan only:** light air-blowing sound.
- **Compressor:** deeper hum/click after a few minutes.

If the fan runs but the compressor never kicks on, likely causes:
- Bad capacitor
- Failed compressor relay
- Unit overheated
- Low refrigerant / sealed-system issue

For a cheap window unit, repair often costs more than replacement.

## 5. Check the back/outside side
Since it is in the window:
- Make sure the **outside rear vents are unobstructed**.
- Make sure the unit tilts slightly **downward outside**, so condensation drains out.
- If it is tilted inward, water can pool and mess with cooling.

## 6. If it blows cold for a bit then stops
Likely:
- Frozen coils
- Dirty filter
- Poor airflow
- Thermostat/Eco mode cycling

Turn it off, run **Fan Only** for 30–60 minutes, clean the filter, then restart on Cool.

## Fast diagnosis

| Symptom | Most likely issue |
|---|---|
| No power at all | Outlet, plug reset, breaker, internal fuse |
| Fan runs, air is warm | Compressor/capacitor/refrigerant issue |
| Blows weak air | Dirty filter or blocked coil |
| Cold then warm | Frozen coil, Eco mode, overheating |
| Water inside | Unit tilted wrong or drain blocked |
| Loud rattling | Loose mounting, fan obstruction, bad bearing |

If it’s blowing air but not cold after reset + filter check + 5 minutes on max cool, it’s probably a compressor/capacitor issue.
