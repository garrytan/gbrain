## Assistant

On Facebook Marketplace, there’s no built-in “archive by item” feature, but here’s how you can handle it when you’ve sold something:

1. **Mark the item as sold**  
   - Go to **Marketplace → Your listings**.  
   - Select the item → choose **Mark as sold** (or **Mark as sold elsewhere**).  
   - This moves the listing out of the active feed.

2. **Messages tied to the listing**  
   - Conversations from that listing will automatically get tagged with the item name and marked “sold” at the top.  
   - They won’t disappear, but they’re grouped under “Marketplace” in Messenger and easier to spot.

3. **Archiving chats manually**  
   - Open Messenger → find the conversation.  
   - On desktop: hover over the chat → click the three dots → **Archive chat**.  
   - On mobile: long-press the chat → select **Archive**.  
   - Repeat for each buyer/seller thread related to that item.

4. **Pro tip**  
   - If you want everything in one place, you can create a custom **“Archived – Sold Items” folder** in Messenger’s search bar by filtering for the item name, then bulk archiving those threads.
