## 🤖 Claude *Agent Teams* (in Claude Code)

**Agent teams** are a new multi-agent feature in Anthropic’s **Claude Code** environment:

- They let you *orchestrate multiple Claude agent sessions* to work together on a shared project in parallel. One agent becomes the “team lead” that coordinates work, divides tasks, and lets teammates communicate directly with each other rather than in a strict master–worker chain. citeturn0search0turn0search15  
- This enables more complex workflows — e.g., research, debugging, large coding features — to be handled concurrently rather than step by step. citeturn0search27  
- Agent teams are currently *experimental/research preview* and need to be enabled in Claude Code settings. citeturn0search27

In summary, agent teams are about **AI collaboration inside Claude Code**, not a user chat interface. citeturn0search0

---

## 📡 Claude + *Microsoft Teams* Integration Options

While *Claude agent teams* is internal to Claude Code, you can *connect Claude with Microsoft Teams* in several ways:

### 🧩 Microsoft 365 Connector (Official)
- Claude can integrate with Teams through Anthropic’s **Microsoft 365 connector** for Team/Enterprise plans. That lets Claude **read and analyze Teams messages and chat content** alongside Outlook, OneDrive, and SharePoint. citeturn0search14turn0search25

### 🛠️ Third-party Automation Tools
Platforms like **Zapier**, **Pipedream**, **Make**, or **Albato** allow you to:
- Trigger Teams messages using Claude outputs.
- Send Claude-generated summaries, alerts or insights into Teams channels automatically via workflows. citeturn0search4turn0search5turn0search10turn0search17

### ⚙️ Developer Integrations (MCP/SDK)
- Developers can connect Claude Code to Teams programmatically using the **Model Context Protocol (MCP)** — letting the AI perform actions like sending chat messages or managing Teams contexts via APIs. citeturn0search32

## 📌 Takeaway

- **Agent teams** are an advanced *multi-agent orchestration feature* within Claude Code — not a direct Microsoft Teams feature. citeturn0search0  
- If you want Claude to *interact with Teams*, you either use the **Microsoft 365 connector** for data analysis and context or third-party automation platforms to *send content into Teams*. citeturn0search14turn0search4
