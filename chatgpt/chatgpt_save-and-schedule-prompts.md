## User

Is there a way i can save prompts in chatgpt atlas browser or schedule the prompt to run again in the future?

---

## Assistant

Mr. White — short answer: yes.

**Save a prompt (re-use it anywhere, including Atlas):**
- **Make a custom GPT** with your prompt baked into its *Instructions* (and optional conversation starters). Then just open that GPT whenever you need it. citeturn5search0turn5search11  
- If you prefer a template library, **Playground → Prompts** lets you save prompt templates (with variables) for repeat use. citeturn5search6  
- For recurring work with files and context, **Projects** keep chats, uploads, and custom instructions together so you can pick up where you left off. citeturn5search4

**Schedule a prompt to run later (one-off or recurring):**
- Use **Tasks in ChatGPT**: “Run this prompt every weekday at 8am and notify me.” Tasks run even if you’re offline and can send push/email, and you can manage them under Settings → Notifications → *Manage tasks* (limit ~10 active). citeturn3view0

**Notes about Atlas:** Atlas is just the browser with ChatGPT built in, so your custom GPTs, Projects, and Tasks work there the same as on web or mobile. It’s currently available on macOS. citeturn6view1

If you tell me the exact prompt and schedule (e.g., “every weekday at 7:30am PT”), I’ll set it up for you now.
