# Jovie Drop Alerts — Landing Page Spec

## Core concept

**Fans sign up once. Jovie alerts them every time an artist drops music, videos, merch, or other releases.**

The page should feel simple, fast, and creator-native. The goal is to make fans understand the value in under 5 seconds and sign up immediately.

---

# 1. Page Goal

Convert fans into subscribers for creator drop alerts.

Primary conversion:

**Fan submits phone number and/or email to receive drop alerts.**

Secondary conversion:

**Fan follows a specific artist/creator on Jovie.**

---

# 2. Target User

A fan who arrives from:

- Artist bio link
- TikTok / Instagram / YouTube description
- QR code at a show
- Merch packaging
- Music video description
- Artist website
- Jovie-hosted creator page

The fan wants to know when the creator releases:

- New music
- New videos
- Merch drops
- Limited editions
- Announcements
- Live events, if added later

---

# 3. Positioning

## One-line positioning

**Join once. Get every drop alert from your favorite artists.**

## Value proposition

Fans should not need to keep checking Instagram, Discord, email, Spotify, YouTube, TikTok, or merch stores. Jovie gives creators one alert channel for every drop.

## Simple user promise

**Sign up one time. Jovie tells you when something drops.**
