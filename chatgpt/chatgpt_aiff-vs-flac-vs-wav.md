## Assistant

Mr. White, here’s a concise overview of using AIFF, FLAC, or WAV on CDJ-3000s:

1. **AIFF**  
   - **Uncompressed, full-quality audio** (same as WAV).  
   - **Metadata-friendly**: Stores titles, album art, cue points neatly.  
   - Slightly larger file sizes than FLAC but provides faster load times than some compressed formats.

2. **FLAC**  
   - **Lossless compression**: Maintains full audio quality at smaller file sizes.  
   - **Metadata support**: Like AIFF, it handles tagging and artwork well.  
   - Minor downside can be slightly slower load times due to on-the-fly decompression, but typically not a major issue on the CDJ-3000.

3. **WAV**  
   - **Uncompressed, full-quality audio** with broad compatibility.  
   - **Limited metadata handling** (less flexible for tagging and artwork).  
   - Large file sizes similar to AIFF, but many DJs use WAV out of habit or for guaranteed playback.

Overall, **AIFF and FLAC** provide better metadata support, which is especially handy in modern digital DJ setups, while **WAV** is a standard fallback for simplicity but lacks robust tagging options.
