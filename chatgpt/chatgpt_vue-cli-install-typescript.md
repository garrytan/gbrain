It seems like the "vue" command is not recognized by your terminal, which could mean that the Vue CLI is not installed on your system. To install it, you can run the following command:

```
npm install -g @vue/cli
```

Once the Vue CLI is installed, you can then run the command `vue add typescript` to add TypeScript support to your Vue.js project.
