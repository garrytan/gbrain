## UI/UX spec
- **Trigger:** “Product” in header opens flyout on hover (desktop, 200ms intent) and on click (mobile/tablet).  
- **Layout (desktop):** 2-column panel like Linear. Left column “Core Features” (top 4). Right column “More” (remaining). Bottom row: slim banner (“What’s new”/Changelog) with subtle divider.  
- **Layout (mobile):** Full-width drawer from top; items stack; large tap targets (min 44px).  
- **Density:** 14–15px body, 600 weight titles, 0.5–0.75rem leading.  
- **Icon treatment:** Circular badge with soft shadow; each icon uses its **own accent var** (no single theme color).  
- **Hover/focus:** Elevate item card (y=2–4), subtle shadow, outline ring using its accent var at 24% opacity.  
- **Animation:**  
  - Panel enter: fade+scale(0.98→1) 120ms; items stagger 20ms.  
  - Respect `prefers-reduced-motion`: disable transforms, keep opacity only.  
- **Close behavior:** click outside, ESC, or moving pointer away for 300ms.  
- **Keyboard:** Trigger is a button with `aria-expanded`. Arrow keys move through items; ENTER opens link; ESC closes; focus returns to trigger.
