## Question

How to make Liquid Glass menu bar in iOS app

## Answer

You do it by layering **blur + tint + rounded shape** behind your buttons, then letting content scroll underneath it.

Below is how to build a “liquid glass” style menu bar in **SwiftUI** (iOS 15+), and then a **UIKit** version.

---

## SwiftUI – Liquid Glass Bottom Menu Bar

```swift
import SwiftUI

struct LiquidGlassMenuBar: View {
    var body: some View {
        ZStack {
            // Your main content
            ScrollView {
                VStack(spacing: 24) {
                    ForEach(0..<40) { i in
                        RoundedRectangle(cornerRadius: 16)
                            .fill(Color.blue.opacity(0.2))
                            .frame(height: 80)
                            .overlay(Text("Row \(i)"))
                            .padding(.horizontal)
                    }
                    .padding(.top, 16)
                }
            }

            // Glass menu bar
            VStack {
                Spacer()
                HStack(spacing: 40) {
                    Image(systemName: "house.fill")
                    Image(systemName: "magnifyingglass")
                    Image(systemName: "heart.fill")
                    Image(systemName: "person.fill")
                }
                .font(.system(size: 18, weight: .semibold))
                .foregroundStyle(.white)
                .frame(height: 60)
                .frame(maxWidth: 320)
                .background(
                    // Glass / liquid material
                    .ultraThinMaterial
                )
                .clipShape(RoundedRectangle(cornerRadius: 26, style: .continuous))
                .overlay(
                    RoundedRectangle(cornerRadius: 26, style: .continuous)
                        .stroke(.white.opacity(0.25), lineWidth: 1)
                )
                .shadow(radius: 18, y: 10)
                .padding(.bottom, 20)
            }
            .ignoresSafeArea(.keyboard, edges: .bottom) // stays above keyboard
        }
    }
}

struct LiquidGlassMenuBar_Previews: PreviewProvider {
    static var previews: some View {
        LiquidGlassMenuBar()
    }
}
```

**Key parts:**

- `.background(.ultraThinMaterial)` → the iOS “glass” blur.
- `clipShape(RoundedRectangle(...))` → soft “liquid” capsule look.
- `.overlay(...stroke...)` → subtle white border like polished glass.
- `.shadow(radius:y:)` → makes it float above content.

You can tweak:

- **Opacity**: add `.opacity(0.9)` on the background for more/less glass.
- **Tint**: layer a color under the material:

```swift
.background(
    ZStack {
        Color.blue.opacity(0.25)
        .ultraThinMaterial
    }
)
```
