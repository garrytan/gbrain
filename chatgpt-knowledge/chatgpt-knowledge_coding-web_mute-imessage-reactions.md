## Question

Yo dawg, you know when someone hearts your message or likes it or whatever on iMessages? Can I make it so that that doesn't count as a notification, so that the badge number doesn't go up just because someone thumbs-upped my fucking message like a douchebag in a 20-person group chat?

## Answer

Yeah, you can quiet those “Tapback” reactions so they don’t trigger a badge.

**iPhone / iPad (iOS 17-18)**  
1. **Settings → Messages.**  
2. Tap **Notifications**.  
3. Under **Alerts**, tap **Customize Notifications** (or “Notifications from People & Apps” on iOS 18).  
4. Scroll to **iMessage Reactions / Tapbacks** and set **Notifications** to **Off** (or choose *Mute Reactions* if that toggle shows).  

If you don’t see that exact wording, use:  
- **Settings → Notifications → Messages → Customize Notifications** → turn **Reactions** or **Mentions & Reactions** off.

That stops likes/hearts from raising the badge or lighting the screen, but you’ll still get real messages normally.
