# Cosine and sine comparison

## Question

Is a cosine technically a sine wave? Is that like a square rambus situation?

## Answer

Yes, cosine is technically a sine wave — just phase-shifted.

Mathematically:
\[
\cos(x) = \sin\left(x + \frac{\pi}{2}\right)
\]

So it’s the same shape, just offset horizontally. Not quite a “square rambus” situation where people use weird branding for the same tech — it’s more like two ways of describing the same waveform depending on what phase you want to start from.
