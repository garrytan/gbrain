# Engine ADRs (AIMS Brain)

Engine-internal design decisions for the `itradeaims-brain` engine live here, in the
iTradeAIMS ADR format (`NNNN-slug.md`, Status/Context/Decision/Consequences). **Network
and identity decisions live in the control plane** (`itradeaims-agent-workflows/docs/adr/`),
not here — e.g. the AIMS Brain ownership decision is ADR-0041 there.
