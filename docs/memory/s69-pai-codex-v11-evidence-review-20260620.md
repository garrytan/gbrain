# s69-PAI-codex V11 evidence-bound review rule

Recorded: 2026-06-20 China time

PAI-codex 一卡评审器必须一一对应 V11 项目评分。评分输出必须先判为什么不能给 S、为什么不能给 A+、是否触发事件驱动降级，再逐项评分 11 项：关系、情绪、成长、留存、付费、爽点密度、反派压迫、戏剧化表达、影视化表达、平台适配、执行风险，最后做关系系数修正和最终评级。

每项评分必须给文章/剧本证据。每个扣分点必须明确扣分内容、扣分原因、对应原文证据和可执行修改建议。评审文件最后必须汇总逐项建议。缺逐项证据、扣分记录或建议闭环的评审，不能用于 A+/S/S+ 判定或 V11 展示导出。
