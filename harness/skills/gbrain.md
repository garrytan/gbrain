# GBrain Skills

## Slugs
- Personal notes: `mem/<topic>`
- Wiki / reference: `wiki/<topic>`
- Always search before writing to avoid duplicates.

## Search modes
- `hybrid` (default): lex + semantic — best for most queries
- `keyword`: exact word match — use for proper nouns, IDs
- `semantic`: concept match — use for fuzzy/conceptual queries

## Write rules
- Slug must start with `mem/` or `wiki/`
- Always include a clear title
- Keep body concise; add `## Sources` section if citing external content
